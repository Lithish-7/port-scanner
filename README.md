# 🔍 Port Scanner

A simple network port scanner written in Python. It checks a host (domain or IP) for open TCP ports.

## 🚀 Features
- Fast socket-based scanning
- Scans custom port ranges
- Command-line interface

## 🛠 Requirements
- Python 3.x

## ▶️ How to Use

1. Run the script:
```bash
python scanner.py
